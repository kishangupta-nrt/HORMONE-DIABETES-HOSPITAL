# HORMONE-DIABETES-HOSPITAL

A premium, modern, and fully responsive static website for **Hormone Diabetes Hospital Raipur & Research Centre**.

## 🌟 Features
- **Compact & Elegant Header Layout:** A sleek sticky navigation bar featuring a high-contrast circular brand logo, perfectly proportioned at all viewports.
- **Mobile-First Responsive Design:** Fully optimized with custom fluid media queries scaling down perfectly on tablets and mobiles (tested across 320px - 1440px+).
- **Comprehensive Hospital Roster & Facilities:** Detailed credentials for all three leading specialists (Dr. Sanjeet Kumar Jaiswal, Dr. Tarun Mishra, Dr. Mrs. Vartika Mishra) and diagnostic departments (Endoscopy, Sonography, Maternity, Laparoscopy).
- **Dual Branch Directory:** Information for both the Shankar Nagar and Choubey Colony branches, complete with interactive Google Maps.
- **SEO & Performance Tuned:** Built with semantic HTML5 elements, Google Fonts, and modern CSS layout architectures.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla custom styling)
- Bootstrap 5.3.3 (Responsive Grid System)
- FontAwesome 6.4.0 (Icons)

## 📂 Project Structure
- `index.html` - Main landing page.
- `assets/css/style.css` - Custom stylesheets containing all design systems and fluid media queries.
- `assets/images/` - Clean images and emblem logos.
